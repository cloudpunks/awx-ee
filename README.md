# awx-ee

[![Docker Build](https://github.com/cloudpunks/awx-ee/workflows/docker/badge.svg)](https://github.com/cloudpunks/awx-ee/actions?query=workflow%3Adocker) [![Upstream Repo](https://img.shields.io/badge/upstream-repo-yellow)](https://github.com/ansible/awx-ee)

This repository is used to build our own custom AWX execution environment with
more current versions of software and collections since the upstream repository
does not provide recent versions.
